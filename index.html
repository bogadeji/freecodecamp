Hi, I'm a developer that loves clean and elegant code.

A little bit about me

Software engineering provides a chance to blend curiosity, creativity, problem-solving skills and continuous learning. My initial foray into software development was driven by curiosity as to what the whole hype around coding and software development entailed.

Getting in has only driven up that curiosity and increased my desire to excel in this career path that I have chosen. My desire to be an excellent, world-class SE has driven my learning and pursuit of learning resources and platforms. I presently live and work from Lagos, Nigeria

Hand-Coded
HTML

My focus is writing clean, well-formated, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate, trouble-shoot and accessible.

Well-Organized
CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.

Ease Converting
Designs into Code

You can trust me to take a design and quickly & accurately convert it into a webpage that is a pixel-perfect match.
